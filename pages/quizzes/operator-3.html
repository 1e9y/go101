What does the following program print? 
package main

const N = 2
var n = N
var a byte = 64 << N / 2
var b byte = 64 << n / 2

func main() {
	println(a, b)
}

Choices:

Answer: 128 0

Run it on Go play.

Key points:

  • Substrings are always non-constant in Go.
  • In a bit-shift expression, if the left operand is an untyped constant and the right operand is not constant, then the type of the left operand will be deduced as the final assumed type. Here, the assumed type is byte.
  • 256 overflows the value range of byte, so it is truncated to 0.
